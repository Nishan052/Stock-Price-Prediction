Detailed Description

Provides ARIMA-based time series forecasting with walk-forward validation.

This module uses pmdarima.auto_arima to determine optimal ARIMA (p, d, q) order, trains ARIMA models on rolling 3-year windows, and forecasts target columns like "Open" and "Close". The most recent model is saved for reuse.

Run walk-forward ARIMA forecasting for each target column.

Parameters
dfTime-indexed DataFrame containing stock data.
columnsList of columns to forecast (e.g., ["Open", "Close"]). Defaults to both if None.
retrainWhether to save the final trained model for reuse. Default: True.
Returns
Dictionary with forecasts (Series) per target column.
  • Uses auto_arima to find the best ARIMA(p,d,q) order on each training window.
  • Each window covers a 3-year period (defined in rollingWindowYears).
  • Forecasts one day ahead for each entry in the test index.
  • Saves the final model to models/arima/arima_model_<column>.pkl.
