Download historical NIFTY 50 data using yFinance.

Data is fetched from January 1, 2008 up to (cutoffDay + 1) to include the latest available data for prediction. The 2008 start date is chosen to ensure a sufficient history while reducing memory and bandwidth load. Only essential columns are retained: Open, High, Low, Close, Volume.
